# `examples/` — Pre-computed pipeline outputs

This directory ships with the public repository so reviewers can see what the
PineSentry-Fire pipeline produces **without running it themselves**. Every file
here was generated by the scripts in `../scripts/` against the data layers in
`../data/` (gitignored, see `STATUS.md` for the inventory).

```
examples/
├── README.md            ← (this file)
├── figures/             ← 16 hero/auxiliary PNG + GIF
├── maps/                ← peninsula atlas + wishlist
└── tables/              ← 27 JSON + 1 CSV — all numerical results
```

## To regenerate any of these locally

```bash
git clone https://github.com/zxsa0716/pinesentry-fire.git
cd pinesentry-fire
pip install -r requirements.txt

# Each example file maps to a specific script — see TABLE.md
python scripts/bootstrap_uncertainty.py        # → bootstrap_5site_95CI.json
python scripts/permutation_test_n1000.py       # → permutation_N1000.json
python scripts/spatial_logit_glmm.py           # → GEE_spatial_logit.json
python scripts/morans_i.py                     # → morans_I_spatial.json
python scripts/boyce_index.py                  # → boyce_continuous.json
python scripts/case_control_sampling.py        # → case_control_1to5.json
python scripts/diff_prospect_torch.py          # → v2_8_DiffPROSPECT_torch.json
python scripts/koflux_nee_validation.py        # → koflux_NEE_dual_validation.json
python scripts/tanager_spectral_ablation.py    # → A1A2_tanager_VNIR_SWIR.json
python scripts/wishlist_priority_score.py      # → wishlist_30_scenes_priority.csv
```

Total reproduction wall-time: ~90 min on a free Colab T4.
